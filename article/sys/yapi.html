    # 安装YApi

    # 环境准备

    • 操作系统：CentOS 7
    • 环境:
      • 安装nodejs
      • 安装Mongodb

    # 安装git

    官方推荐，在内网部署的时候安装git，可以根据自己所需安装

    yum -y install git

    # 搭建YApi环境

    准备环境搭建完成后，开始搭建YApi，安装命令

    npm install -g yapi-cli --registry https://registry.npm.taobao.org

    启动可视化部署程序

    yapi server
