    # 安装pm2及使用

    # 安装pm2

    npm install -g pm2 --registry https://registry.npm.taobao.org

    # 查看当前版本

    pm2 -v

    # 运行

    pm2 -v app.js --name app

    app.js是项目启动文件，app是自定义的进程名

    # 停止进程

    pm2 stop app

    # 开机启动

    • 运行pm2 startup，即在/etc/init.d/目录下生成pm2-root的启动脚本，且自动将pm2-root设为服务。
    • 运行pm2 save，会将当前pm2所运行的应用保存在/root/.pm2/dump.pm2下，当开机重启时，运行pm2-root服务脚本，并且到/root/.pm2/dump.pm2下读取应用并启动。
